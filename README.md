# 📄 GraphRAG para Análise Inteligente de Contratos

Aplicação web de Inteligência Artificial para análise de contratos em tempo real utilizando GraphRAG e modelos de linguagem (LLMs). O sistema combina Retrieval-Augmented Generation (RAG) com grafos de conhecimento para melhorar a precisão das respostas e permitir consultas inteligentes sobre documentos jurídicos.

---

## 🚀 Visão Geral

O projeto implementa uma arquitetura de Inteligência Aumentada capaz de:

- Processar contratos em PDF
- Extrair entidades e relações relevantes
- Construir um grafo de conhecimento
- Responder perguntas sobre o contrato com alta precisão
- Utilizar LLMs para geração de respostas contextualizadas

A solução demonstra como a combinação de RAG com grafos de conhecimento pode superar limitações de sistemas tradicionais baseados apenas em busca vetorial.

---
